🏥 Hospital Management System (ML-Enabled)

A full-stack Hospital Management System (HMS) integrated with Machine Learning and NLP, designed to improve healthcare operations, patient care, and decision-making through intelligent automation.

🚀 Overview

This system goes beyond traditional HMS platforms by incorporating predictive analytics and AI-driven features such as:

🤖 Chatbot-based patient interaction
🧠 Disease prediction
⚠️ Patient readmission risk prediction
📉 Appointment no-show prediction

It provides a secure, role-based platform for Admins, Doctors, and Patients to manage hospital workflows efficiently.

✨ Features
🔐 Authentication & Authorization
JWT-based secure authentication
Role-based access control (Admin / Doctor / Patient)
🧑‍⚕️ Patient Management
Patient registration & profile management
Medical history tracking
🩺 Doctor Module
Manage availability
View appointments
Patient interaction
📅 Appointment System
Book appointments based on real-time availability
Prevent duplicate bookings
Track appointment history
📊 Admin Dashboard
View hospital statistics
Monitor doctors, patients, and appointments
🤖 Machine Learning Features
🧠 Disease Prediction
Predicts possible diseases based on patient symptoms
Built using Scikit-learn models
⚠️ Readmission Risk Prediction
Identifies patients at risk of readmission
Helps in proactive care planning
📉 No-show Prediction
Predicts likelihood of missed appointments
Improves scheduling efficiency
💬 NLP Chatbot
Interactive chatbot for symptom-based Q&A
Guides patients through diagnosis flow
